# stml-koa

This package is a middleware which connects koa view engine to stml.

## Install

``` bash
$ npm install koa-stml
# Or
$ yarn add koa-stml
```

## Usage

``` js
var render = require('stml-koa')

render(app, {
  ext: 'html'
})

```
